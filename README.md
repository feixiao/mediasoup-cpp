# mediasoup-cmake

1. use cmake instead of gyp.
2. use cpp instead of js.

About [mediasoup](https://github.com/versatica/mediasoup), you can visit [here](https://github.com/versatica/mediasoup) to learn more.

#### Build

```shell
mkdir build && cd build
cmake ../
make -j 4
```

#### System

- OSX 11.2.3
- Ubuntu 20.04 LTS
